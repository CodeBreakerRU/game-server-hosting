DirectX® 9.0c

During the games setup, the install process will prompt you to install Microsoft® DirectX on your computer.  If you choose "Yes", the Microsoft DirectX setup program will install the appropriate files on your computer.

Please refer to the Microsoft DirectX website for further information or localization updates.

http://www.microsoft.com/directx

Q. What is DirectX 9.0c and why do I need it?

A. DirectX 9.0c is a set of sound, video, input and network functions developed by Microsoft. Many games including this one utilize the functions that are included with DirectX 9.0c.

Q. If I don't install DirectX 9.0c when I install the game, can I install it later?

A. Yes. You will need to manually install it. To do so, use the following instructions:

  1. Place Disc 1 in the CD/DVD-Rom drive. The Autoplay Splash Screen should appear. (See the Autoplay section of the help documentation if it does not appear.)

  2. Click on the "Extras" button.

  3. Click on the "Install DirectX" and follow the installation process.  When the DirectX installation finishes, you may need to restart your computer for the full functionality of DirectX to take effect.

Q. I already have other Windows® 98/98SE/ME/2000/XP/VistaTM games installed on my computer.  Will Microsoft DirectX installer change my DirectX files?

A. If you already have other Windows 98/98SE/ME/2000/XP/Vista games on your computer, chances are you already have a version of DirectX installed. In this case, the DirectX installer will detect and overwrite any previous versions with DirectX 9.0c. At that point, you may need to restart your computer for the full functionality of DirectX to take effect. If you already have DirectX 9.0c installed on your computer, the Microsoft DirectX 9.0c installer will detect it and not overwrite any files. You will not need to restart your computer after installation if no files were installed or overwritten.

Vista Note: Windows Vista comes with DirectX 10 installed. To play pre-DirectX 10 games, DirectX 9.0c must be installed. (This will not adversely effect game play)
